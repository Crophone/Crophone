D. Divisible Pairs

here is the O(n) code.
if ($a_i$ + $a_j$) % x = 0,then $a_i$%x = (x-$a_j$%x)%x
for example, (2+4)%6=0,then 2%6=(6-4%6)%6
if ($a_i$ - $a_j$) % y = 0,then $a_i$%y = $a_j$%y

then if $a_i$=1,it find all $a_j$=5,according to the code
but if $a_i$=3,it will find itself one more time.

here we should notice the repetition, ($a_i$+$a_i$)%x=0
